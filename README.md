# Real-Time Dashboard (Streamlit)

## Description
A real-time sales dashboard built using Python and Streamlit. It displays sales data, interactive charts, and key business metrics.

## Features
- Display sales data from CSV
- Interactive line chart
- Bar chart for profit
- Total Sales metric
- Total Profit metric
- Refresh Dashboard button

## Technologies Used
- Python
- Streamlit
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
